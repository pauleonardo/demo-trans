# Demo Transacciones

___

##### Este repositorio contiene el código de un proyecto básico, basado en un administrador de un sistema de transacciones.
___
### **Funcionalidades:**
- Crear Usuario
- Editar Usuario
- Eliminar Usuario
- Aperturar cuenta de Usuario
- Listar Usuarios
- Lista Transacciones de Usuario
- Transeferir a otro Usuario
